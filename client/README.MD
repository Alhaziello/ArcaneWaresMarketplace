## Getting Started

How to use this website
To use this website, you need to register and create an account. Once you have registered, you can browse the catalog of magical items for sale, and get in touch securely with other providers. You can also sell your own magical items by creating a listing and setting your own price.

## Available Scripts

### Running the development server.

```bash
    npm run dev
```

### Building for production.

```bash
    npm run build
```

### Running the production server.

```bash
    npm run start
```
